# CampusCare – Emergency Medical Alert System

## SDG Chosen
SDG 3 – Good Health & Well-Being

## Team Members
- Afeq – Design & Prototype Lead
- Jairaj – Planning, UML & Agile Lead

## Project Description
CampusCare is a mobile-based emergency medical alert system designed for university campuses. The system allows students to trigger an emergency alert with one tap, automatically sharing their location with campus clinic staff for faster response and improved safety.

## Target Users
- Students
- Clinic Staff
- Admin

## Core Features
- One-tap emergency button
- Automatic location sharing
- Real-time clinic notification
- Emergency status tracking
- First aid guidance
- Staff dashboard

## SDLC & Development Approach
This project follows Agile methodology using Scrum practices. Development is divided into two sprints:
- Sprint 1: Planning, UML analysis, and prototype design
- Sprint 2: Testing, improvement, and refinement

Project tasks are managed using Trello, and progress is tracked through GitHub commits.
